엣지 컴퓨팅

엣지 컴퓨팅(Edge Computing)은 클라우드 컴퓨팅과 반대되는 개념으로,

인터넷이 아닌 로컬 장치(예: 스마트폰, 태블릿, IoT 장치 등)에서 데이터를 처리하는 기술입니다.

이를 통해 데이터 처리 및 분석이 인터넷 대역폭을 절약하고, 응답 시간을 단축하여 네트워크 대역폭 혼잡을 완화할 수 있습니다.

엣지 컴퓨팅을 사용하는 이유

대부분의 기업에서는 일반적으로 퍼블릭 클라우드나 프라이빗 클라우드 환경의 중앙 저장소에 데이터를 저장하여 관리하고 분석합니다.

하지만 기존 인프라와 클라우드 컴퓨팅은 더이상 여러 어플리케이션의 요구 사항을 충족하지 못합니다.

예를 들어, IoT(사물인터넷)와 IoE(만물인터넷)의 경우 대량의 데이터를 실시간으로 처리하려면 지연이 최소화되고 가용성이 뛰어난 네트워크가 필요하지만, 기존 IT 인프라로는 불가능합니다.

이 경우 엣지 컴퓨팅의 장점이 더욱 확실해집니다.

엣지 컴퓨팅의 장점

  • 성능 향상

    • 중앙집중형 데이터 센터나 클라우드로 수집한 데이터를 업로드하는 대신 데이터가 생성되는 위치에서 컴퓨팅 작업을 진행하면 지연시간을 줄일 수 있습니다. 또한 지능형 엣지 컴퓨팅 기술을 활용하면 항상 클라우드와 연결을 유지할 필요가 없기에 장치의 배터리를 절약하고 성능을 높이는 효과도 있습니다.

    엣지 컴퓨팅의 성능 향상 효과는 이전에는 불가능했던 다양한 작업을 가능하게 합니다. 예를 들어 5G 연결로 도시의 인프라와 서비스를 관리하는 스마트 시티를 구현하려면 커뮤니케이션, 위치 추적, 교통 통제 등 다양한 데이터를 실시간으로 관리해야 합니다.

  • 서비스 유연성 확보 가능

    • 기존 클라우드 컴퓨팅 기술을 활용하기 위해서는 항상 인터넷에 연결된 상태를 유지해야 합니다. 하지만 엣지 컴퓨팅은 인터넷 연결 없이도 데이터를 처리할 수 있기에 인터넷 연결이 어렵거나 불안정한 위치에서도 컴퓨팅 작업을 진행할 수 있어 업무와 서비스의 유연성이 높아집니다.

    엣지 컴퓨팅은 네트워크 레이턴시를 줄이고 대역폭을 확보하는 역할을 합니다. 또한 현장에서 판단이 필요한 데이터는 신속하게 분석해 적절한 결정을 내리고, 심층 분석이 필요한 데이터는 중앙에서 관리하도록 할 수 있습니다.

  • 인공 지능 및 기계 학습에 적합

    • 인공 지능과 기계 학습을 위해서는 방대한 양의 데이터가 필요하며, 이렇게 방대한 양의 데이터를 클라우드로 전송하려면 많은 비용과 시간과 처리 성능이 필요합니다. 엣지 컴퓨팅은 활용하면 장치와 센서에서 생성된 데이터를 그 자리에서 신속하게 처리할 수 있기에 인공 지능과 기계 학습 관련 서비스에 적합합니다.

    엣지 컴퓨팅은 스마트 공장 등 인공 지능과 기계 학습이 적극 활용되는 분야에서 특히 유용합니다. 예를 들어 스마트 공장은 엣지 컴퓨팅을 통해 공장에서 발생하는 데이터를 그 자리에서 수집하고 처리하며, 데이터를 클라우드로 전송하고 분석을 기다리는 대신 현장에서 즉시 적절한 조치를 취할 수 있습니다.

    엣지 컴퓨팅 적용 분야

  • 스마트 홈 및 스마트 빌딩

    • 엣지 컴퓨팅 기술을 이용하여 스마트 홈 및 스마트 빌딩 시스템을 구축할 수 있습니다.

    이를 통해 건물 내부 및 외부 환경 모니터링, 에너지 효율성 향상, 보안 강화 등 다양한 기능을 구현할 수 있습니다.

  • 자율 주행 자동차

    • 로컬에서 데이터 처리가 이루어지기 때문에 자동차 내부에서 데이터 처리 및 분석이 이루어지며, 이를 통해 자율 주행 및 안전성을 향상시킬 수 있습니다.

  • 제조 산업

    • 제조 산업에서는 엣지 컴퓨팅 기술을 이용하여 공정 모니터링, 예측 유지보수, 기계 학습 및 분석 등을 수행할 수 있습니다.

    이를 통해 생산성을 향상시키고, 생산 라인 내 장애물을 예방할 수 있습니다.

  • 의료 분야

    • 로컬에서 데이터 처리가 이루어지기 때문에 의료 장비에서 발생하는 데이터를 빠르게 처리하고 분석할 수 있습니다.

    이를 통해 질병 예방, 진단, 치료 등의 분야에서 혁신적인 솔루션을 구현할 수 있습니다.

  • 에너지 분야

    • 에너지 데이터를 수집하고 분석함으로써 에너지 효율성을 높일 수 있습니다.

    클라우드 컴퓨팅과 엣지컴퓨팅의 차이

    1. 위치

      • 클라우드 컴퓨팅: 데이터 처리와 저장이 중앙 집중화된 데이터 센터나 클라우드 서버에서 이루어집니다.

      • 엣지 컴퓨팅: 데이터 처리와 저장이 데이터를 생성하는 장소와 더 가까운 '가장자리'에서 이루어집니다. 이 가장자리는 주로 사용자 또는 데이터를 생성하는 디바이스 근처에 위치한 로컬 서버, 게이트웨이, 혹은 IoT장치 등을 의미합니다.

    2. 지연 시간

      • 클라우드 컴퓨팅: 데이터가 중앙 집중화된 서버로 이동해야 하므로 전송에 시간이 소요되며, 이로 인해 더 긴 응답 시간이 발생할 수 있습니다.

      • 엣지 컴퓨팅: 데이터 처리가 가장자리에서 이루어지므로 데이터 전송에 소요되는 시간이 줄어들어 빠른 응답 시간을 제공합니다.

    3. 보안

      • 클라우드 컴퓨팅: 데이터가 클라우드로 전송되므로 데이터 보안에 대한 우려가 있을 수 있습니다.

      • 엣지 컴퓨팅: 데이터가 로컬에서 처리되므로 데이터 보안이 강화될 수 있습니다.

    4. 응용 분야

      • 클라우드 컴퓨팅: 대규모 데이터 처리와 분석, 웹 어플리케이션 호스팅, 데이터 저장 등의 분야에 주로 사용됩니다.

      • 엣지 컴퓨팅: IoT기기, 스마트 시티, 산업 자동화, 헬스케어 등 실시간 데이터 처리와 응용이 필요한 분야에서 주로 사용 됩니다.

    5. 요약

      6. 클라우드 컴퓨팅은 중앙 집중화된 데이터 센터나 클라우드 서버에서 데이터 처리를 하지만, 엣지 컴퓨팅은 데이터를 생성하는 장소와 더 가까운 가장자리에서 데이터 처리를 합니다. 이로 인해, 응답 시간이 줄어들고 대역폭을 절약하며, 데이터 보안을 강화할 수 있습니다.

    인텔

    인텔이 MWC24에서 기업 고객이 클라우드처럼 간편하게 대규모로 엣지 및 AI 애플리케이션을 개발·배포·실행·보안·관리하도록 지원하는 모듈식 개방형 소프트웨어 플랫폼인 새로운 ‘엣지 플랫폼(Edge Platform)’을 발표했습니다.

    엣지 컴퓨팅 기술의 발전과 적용을 통해 더 나은 AI 플랫폼을 이용할 수 있습니다.

    분산형 GPU 인프라 기업 에이더(Aethir)가 엣지 컴퓨팅 기기인 ‘에이더 엣지(Aethir Edge)’를 5월 23일 출시한다고 발표했습니다.

    에이더 엣지는 고성능 엣지 AI 장치를 분산형 클라우드 네트워크에 연결하여 원활한 병렬 처리와 안정적인 가동 시간을 보장합니다.

    이를 통해 컴퓨팅 파워가 특정 위치나 엔티티에 국한되지 않도록 합니다.
